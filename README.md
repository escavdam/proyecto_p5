# P5 Template

Template para proyectos con p5.js con Express y Node.js.

## Instalación

1. Clonar el repositorio
```bash
git clone
```

2. Instalar dependencias
```bash
npm install
```

3. Lanzar el servidor
```bash
npm start
```

4. Abrir el navegador en `http://localhost:3000`
